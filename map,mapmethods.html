Map : is a data structure which stores values as key-value pairs,here keys can be of anytypes (Map).

Methods in Map :-

set => is used to add keys and values to a Map.

get => is used to access the value using its key in a Map.

has => is used to check if a key is within the Map.

delete => is used to delete the values based on a key in a Map.
