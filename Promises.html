Promises: -

Promise is an object to handle async functionality in Javascript.

-> Promise has 3 different states

1. Pending

2. Fulfill

3. Reject

API :- Application Programming Interface , is the heart of the web application that acts as the intermediate connection between frontend and the database to carry data.

-> API are of two types :

1. SOAP - Simple Object Access Protocol. => deals with XML .

2. REST - Representational State Transfer. => deals with XML,JSON,TEXT,HYPERTEXT.



Methods in promises : -

-> then() : when the promises is successful (or) fulfilled.

-> catch() : when the promises is rejected.

-> finally() : It will be executed even when the promise is fulfilled or rejected at the end.
