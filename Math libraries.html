Math Library : used for performing mathematical operations.

Math.pow(x,y) => x power of y.
Math.sqrt() => square root of number.
Math.floor() => decreased value.
Math.ciel() => increased value.
Math.round() => increase or decrease based on value (0.5).
Math.max() => maximum value. Math.min() => minimum value.
Math.sin() => sine function.
Math.random() => generates a value between 0 and 1 randomly.
