Callback : Callback is a function which is passed to another function as a parameter.

-> Callback is used to handle async functionality in Javascript.

-> The function which accepts another function as parameter is called as higher order function.

* setTimeout is a best example of callback.

** A function which expects a function as an argument/parameter is known as higher order function.

Syntax :

function A(myfun){

}

function B{

}

A(B); //A- higher order function & B - is a callback .
